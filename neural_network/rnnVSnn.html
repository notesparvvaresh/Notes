Neural Networks (NNs)

Also known as feedforward neural networks (FFNNs).

  • Consist of an input layer, one or more hidden layers, and an output layer.
  • Information flows in one direction, from input to output.
  • Suitable for tasks where input and output are independent or lack a sequential relationship.
  • Examples include image classification, regression tasks, etc.

Recurrent Neural Networks (RNNs)

Designed to handle sequential data where:

  • The current input depends on both the current state and past states (context).
  • Have connections forming cycles, exhibiting temporal dynamic behavior.
  • Each neuron maintains an internal state as memory to remember past information.
  • Suitable for tasks such as time series prediction, speech recognition, language modeling, etc.
  • However, vanilla RNNs suffer from issues like vanishing gradients and difficulty in learning long-term dependencies.

Artificial Neural Networks (ANNs)

General term encompassing both NNs and RNNs, but typically refers to:

  • The broader class of machine learning models inspired by biological neurons.
  • Includes architectures like CNNs, GANs, etc., in addition to NNs and RNNs.
  • Used across various domains for pattern recognition, decision-making, and more.
