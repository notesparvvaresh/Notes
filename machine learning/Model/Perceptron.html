Perceptron Algorithm

The Perceptron algorithm is a supervised learning algorithm used for binary classifiers, which are models that can decide whether an input, represented by a vector of numbers, belongs to one class or another. Here is a step-by-step outline of the Perceptron algorithm:

Perceptron Algorithm

  1. Initialize the weights and bias

    Initialize the weights w to zero or small random numbers.

    Initialize the bias b to zero or a small random number.

  2. Define the activation function

    The activation function is usually a step function that returns +1 if the input is greater than or equal to zero, and -1 otherwise.

  3. Training the Perceptron

    Repeat until convergence or for a fixed number of epochs:

    1. For each training example (x_i, y_i):

      Calculate the output y_hat using the activation function:
      y_hat = sign(w · x_i + b)
      Update the weights and bias if there is a misclassification (i.e., y_hat is not equal to y_i):
      w ← w + η(y_i - y_hat) x_i
      b ← b + η(y_i - y_hat)

      Here, η is the learning rate.
